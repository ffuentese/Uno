Este es el juego Uno de cartas pero en modo texto. No debe considerarse una versión definitiva.

## Requerimientos

Sólo tener Python 3 para ejecutarlo.

## Instrucciones

- Tener Python 3 instalado
- Bajar ambos archivos.
- Ejecutar python Uno.py (si aparece un error de sintaxis inválida es que están ejecutando con Python 2)

El usuario es responsable de su mazo de cartas y la implementación de las reglas está basada en las 
[reglas oficiales](http://www.uno-juego.es/reglas-del-juego/) excepto lo de gritar UNO al quedar con sólo una carta.
 
 - Cada usuario recibe 7 cartas
 - En cada jugada el jugador tiene la opción de pedir una carta. Si no ha tomado y decide pasar, el programa le pasa una carta.
 - Si el jugador es el blanco de una carta especial como +2 y +4 pierde su turno.
 - No es necesario gritar UNO!  
 - Cuando el usuario pierde un turno por cualquier carta especial, el juego detiene su ejecución para asegurarse 
 que el usuario se dio cuenta.
 - Las jugadas de los bots toman 5 segundos para que el usuario pueda ver la progresión del juego 
 
 En la medida que tenga tiempo para mejorarlo lo haré pero como curiosidad lo dejaré aquí. Lo más complejo en mi opinión ha sido el "ciclo de juego" (los turnos) 
 por las particularidades de las reglas (no basta con el cycle). Ahí y sólo ahí me apoyé del [código de Ben Nuttall](https://github.com/bennuttall/uno) 
 y aún así tuve que adaptar su "reversedcycle".
